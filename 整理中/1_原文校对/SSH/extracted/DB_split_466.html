18.4.1 Choosing a Key

To select the private key for PuTTY to use, open the PuTTY Configuration dialog and visit Connection/SSH/Auth. Under "Private key file for authentication," browse to and select your key of choice. (Make sure the corresponding public key is properly installed on the server. [6.2])

If you're using Plink, you can choose the key on the command line with the -i option:

    C:\> plink -i c:\keys\me.ppk smith@server.example.com
