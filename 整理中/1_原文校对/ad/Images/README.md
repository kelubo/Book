# 图片使用说明

## 图片列表

本目录需要包含以下图片文件：

1. **male_reproductive_system** - 男性生殖系统解剖图
2. **female_reproductive_system** - 女性生殖系统解剖图
3. **sexual_response_cycle** - 性反应周期示意图（马斯特斯和约翰逊模型）
4. **condom_usage** - 男用避孕套正确使用步骤示意图
5. **std_prevention** - 性传播疾病预防措施示意图
6. **breast_self_exam** - 乳腺自我检查步骤示意图

## 图片格式要求

- 建议使用 **JPG** 或 **PNG** 格式
- 分辨率建议不低于 **300 DPI**，以保证打印质量
- 图片尺寸适中，建议宽度在 **1000-1500像素** 之间

## 图片内容建议

### 1. 男性生殖系统解剖图
- 清晰展示男性外生殖器和内生殖器结构
- 标注主要器官名称（如阴茎、阴囊、睾丸、附睾、输精管、精囊腺、前列腺等）
- 使用正面和侧面视图，便于理解

### 2. 女性生殖系统解剖图
- 清晰展示女性外生殖器和内生殖器结构
- 标注主要器官名称（如阴阜、大阴唇、小阴唇、阴蒂、阴道、子宫、卵巢、输卵管等）
- 使用正面和侧面视图，便于理解

### 3. 性反应周期示意图
- 展示马斯特斯和约翰逊提出的四个阶段：兴奋期、平台期、高潮期、消退期
- 分别展示男性和女性的性反应曲线对比
- 标注各阶段的主要生理变化

### 4. 男用避孕套正确使用步骤示意图
- 分步骤展示避孕套的正确使用方法
- 包括：检查包装、打开包装、捏紧储精囊、佩戴避孕套、性交后取出等步骤
- 使用简洁明了的图示和文字说明

### 5. 性传播疾病预防措施示意图
- 展示主要的性传播疾病预防方法
- 包括：使用避孕套、限制性伴侣数量、定期检查、接种疫苗等
- 使用图标和文字说明相结合的方式

### 6. 乳腺自我检查步骤示意图
- 分步骤展示乳腺自我检查的正确方法
- 包括：站立位观察、平卧位触摸、挤压乳头等步骤
- 标注正确的触摸手法和检查范围

## 注意事项

- 图片内容应科学、准确，避免误导
- 避免使用过于暴露或不适当的图片
- 确保图片版权合法，建议使用原创或无版权图片
- 图片文件名称应与文档中的引用名称完全一致（包括大小写）

## 图片引用方式

在LaTeX文档中，图片通过以下方式引用：

```latex
\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.7\linewidth]{图片文件名}
    \caption{图片说明}
    \label{fig:图片标签}
\end{figure}
```

例如：

```latex
\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.7\linewidth]{male_reproductive_system}
    \caption{男性生殖系统解剖图}
    \label{fig:male_reproductive_system}
\end{figure}
```

请确保图片文件名称与上述代码中的"图片文件名"完全一致，包括扩展名（如.jpg或.png）。
