\chapter{常用测量仪器}
\label{chap:common_instruments}

\section{示波器}
\label{sec:oscilloscope}

示波器是一种用于观察和测量电信号波形的仪器。

\subsection{工作原理}

示波器通过高速电子束在荧光屏上扫描，将电信号转换为可见的波形。

\subsection{主要功能}
\begin{itemize}
    \item \textbf{波形显示}：观察信号的形状和变化
    \item \textbf{电压测量}：测量信号的幅度和峰值
    \item \textbf{频率测量}：测量信号的频率和周期
    \item \textbf{相位测量}：测量两个信号之间的相位差
    \item \textbf{脉冲测量}：测量脉冲的上升时间、下降时间等参数
\end{itemize}

\subsection{类型}
\begin{itemize}
    \item \textbf{模拟示波器}：基于阴极射线管工作
    \item \textbf{数字示波器}：将信号数字化后显示
    \item \textbf{存储示波器}：可以存储和回放波形
    \item \textbf{数字荧光示波器}：具有高刷新率和波形累积功能
\end{itemize}

\section{信号发生器}
\label{sec:signal_generator}

信号发生器是用于产生各种波形电信号的仪器。

\subsection{主要功能}
\begin{itemize}
    \item \textbf{波形产生}：产生正弦波、方波、三角波等波形
    \item \textbf{频率调节}：调节输出信号的频率
    \item \textbf{幅度调节}：调节输出信号的幅度
    \item \textbf{调制功能}：实现调幅、调频等调制方式
\end{itemize}

\subsection{类型}
\begin{itemize}
    \item \textbf{函数发生器}：产生标准函数波形
    \item \textbf{任意波形发生器}：可以产生自定义波形
    \item \textbf{信号源}：用于提供特定频率和幅度的信号
    \item \textbf{脉冲发生器}：产生脉冲信号
\end{itemize}

\section{经典学生信号源 J2465}
\label{sec:student_signal_source}

J2465型学生信号源是20世纪80年代中国广泛使用的教学仪器，由新金师范教学仪器厂生产。它是电子技术课程中不可或缺的实验设备，为一代电子爱好者和工科学生提供了入门学习的工具。

\subsection{技术参数}

\begin{itemize}
    \item \textbf{频率范围}：10 Hz ~ 1 MHz（分多个频段）
    \item \textbf{输出波形}：正弦波、方波、三角波
    \item \textbf{输出幅度}：0 ~ 5 V（连续可调）
    \item \textbf{输出阻抗}：600 Ω
    \item \textbf{电源}：220 V AC
\end{itemize}

\subsection{面板结构}

J2465的控制面板包含以下主要部件：

\begin{itemize}
    \item \textbf{频率选择开关}：选择不同的频率范围
    \item \textbf{频率微调旋钮}：在选定范围内精确调节频率
    \item \textbf{波形选择开关}：选择正弦波、方波或三角波
    \item \textbf{幅度调节旋钮}：调节输出信号的幅度
    \item \textbf{输出端子}：提供信号输出接口
    \item \textbf{电源开关}：控制仪器电源
\end{itemize}

\subsection{使用方法}

使用J2465学生信号源的基本步骤：

\begin{enumerate}
    \item 将仪器连接到220V交流电源
    \item 根据实验需求选择合适的波形
    \item 设置频率范围并微调至所需频率
    \item 调节输出幅度至合适大小
    \item 将输出端子连接到被测电路或示波器
\end{enumerate}

\subsection{历史意义}

J2465学生信号源在中国电子教育史上具有重要意义：

\begin{itemize}
    \item \textbf{普及教育}：为广大中学生和大学生提供了接触电子技术的机会
    \item \textbf{培养人才}：许多电子工程师和无线电爱好者都是通过它入门的
    \item \textbf{历史价值}：见证了中国电子工业从起步到发展的历程
\end{itemize}

\section{万用表}
\label{sec:multimeter}

万用表是一种多功能测量仪器，可以测量电压、电流、电阻等多种电量。

\subsection{主要功能}
\begin{itemize}
    \item \textbf{电压测量}：测量直流和交流电压
    \item \textbf{电流测量}：测量直流和交流电流
    \item \textbf{电阻测量}：测量电阻值
    \item \textbf{二极管测试}：检测二极管的好坏
    \item \textbf{通断测试}：检测电路是否导通
\end{itemize}

\subsection{类型}
\begin{itemize}
    \item \textbf{模拟万用表}：使用指针显示测量结果
    \item \textbf{数字万用表}：使用数字显示测量结果
    \item \textbf{钳形表}：可以测量交流电流而不需要断开电路
    \item \textbf{兆欧表}：用于测量高电阻
\end{itemize}

\section{频率计}
\label{sec:frequency_counter}

频率计是用于测量信号频率的仪器。

\subsection{工作原理}

频率计通过计数一定时间内信号的周期数来计算频率。

\subsection{主要功能}
\begin{itemize}
    \item \textbf{频率测量}：测量信号的频率
    \item \textbf{周期测量}：测量信号的周期
    \item \textbf{计数功能}：对脉冲信号进行计数
    \item \textbf{时间间隔测量}：测量两个信号之间的时间间隔
\end{itemize}
