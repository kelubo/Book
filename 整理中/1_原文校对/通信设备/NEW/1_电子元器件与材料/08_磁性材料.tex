\chapter{磁性材料}
\label{chap:magnetic_materials}

\section{磁性材料基础}
\label{sec:magnetic_fundamentals}

磁性材料是指具有铁磁性或亚铁磁性的材料，其磁导率远大于1。磁性材料的主要性能参数包括：

\begin{itemize}
    \item \textbf{磁导率}：$\mu = \frac{B}{H}$，表示材料磁化的难易程度
    \item \textbf{饱和磁感应强度}：$B_s$，材料能达到的最大磁感应强度
    \item \textbf{矫顽力}：$H_c$，使材料退磁所需的磁场强度
    \item \textbf{剩磁}：$B_r$，外磁场消失后材料保留的磁感应强度
\end{itemize}

\section{软磁材料}
\label{sec:soft_magnetic_materials}

软磁材料具有低矫顽力和高磁导率，容易磁化和退磁。

\begin{itemize}
    \item \textbf{特点}：$H_c < 100$ A/m，磁导率高，损耗低
    \item \textbf{种类}：纯铁、硅钢片、坡莫合金、软磁铁氧体
    \item \textbf{应用}：变压器铁芯、电感磁芯、电机定子
\end{itemize}

硅钢片是最常用的软磁材料，含硅量约3-5%，具有低铁损特性。

\section{硬磁材料}
\label{sec:hard_magnetic_materials}

硬磁材料具有高矫顽力和高剩磁，磁化后能长期保持磁性。

\begin{itemize}
    \item \textbf{特点}：$H_c > 100$ A/m，磁能积$(BH)_{max}$大
    \item \textbf{种类}：铝镍钴合金、铁氧体永磁、稀土永磁（NdFeB、SmCo）
    \item \textbf{应用}：永磁电机、扬声器、磁选设备
\end{itemize}

稀土永磁材料的磁能积可达$400$ kJ/m³以上，是目前性能最好的永磁材料。

\section{铁氧体材料}
\label{sec:ferrite_materials}

铁氧体是一种陶瓷磁性材料，由氧化铁和其他金属氧化物组成。

\begin{itemize}
    \item \textbf{软磁铁氧体}：如Mn-Zn铁氧体、Ni-Zn铁氧体，用于高频变压器、滤波器
    \item \textbf{硬磁铁氧体}：如钡铁氧体、锶铁氧体，用于永磁电机、磁钢
\end{itemize}

铁氧体具有高电阻率，适合高频应用，但饱和磁感应强度较低。

\section{磁性元件应用}
\label{sec:magnetic_component_applications}

磁性材料在电子设备中应用广泛：

\begin{itemize}
    \item \textbf{变压器}：利用互感原理实现电压变换
    \item \textbf{电感器}：储存磁场能量，用于滤波、谐振
    \item \textbf{磁头}：用于读写磁介质信息
    \item \textbf{磁屏蔽}：防止磁场干扰
\end{itemize}
