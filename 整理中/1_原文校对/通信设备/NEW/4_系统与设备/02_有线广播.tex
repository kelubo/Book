\chapter{有线广播}
\label{chap:wired_broadcast}

\section{有线广播概述}
\label{sec:wired_broadcast_intro}

有线广播是通过有线传输介质（如电缆、双绞线）传输音频信号的广播系统。与无线广播相比，有线广播具有信号稳定、抗干扰能力强、音质好等优点。

有线广播的应用场景：
\begin{itemize}
    \item \textbf{公共广播}：学校、商场、车站、机场等公共场所
    \item \textbf{应急广播}：火灾报警、紧急通知等
    \item \textbf{校园广播}：学校教学广播、作息铃声
    \item \textbf{楼宇广播}：背景音乐、通知广播
\end{itemize}

\section{有线广播系统组成}
\label{sec:wired_broadcast_components}

有线广播系统的基本组成：

\begin{itemize}
    \item \textbf{音源设备}：麦克风、CD播放器、MP3播放器、调音台
    \item \textbf{功率放大器}：将音频信号放大到足够功率
    \item \textbf{传输线路}：音频电缆、双绞线、光纤
    \item \textbf{终端设备}：扬声器、音柱、壁挂音箱、号角喇叭
    \item \textbf{控制设备}：分区控制器、定时器、远程控制
\end{itemize}

\section{有线广播传输技术}
\label{sec:wired_broadcast_transmission}

有线广播的传输方式：

\begin{itemize}
    \item \textbf{定压传输}：常用70V或100V定压输出，适合远距离传输
    \item \textbf{定阻传输}：适合短距离、高保真传输
    \item \textbf{数字传输}：采用IP网络传输，支持多通道、远程控制
    \item \textbf{光纤传输}：适合长距离、抗干扰要求高的场合
\end{itemize}

定压传输的优点：线路损耗小，便于多个扬声器并联使用。

\section{有线广播设备}
\label{sec:wired_broadcast_equipment}

主要设备介绍：

\begin{itemize}
    \item \textbf{功率放大器}：定压功放、定阻功放、数字功放
    \item \textbf{扬声器}：吸顶扬声器、壁挂扬声器、音柱、号角喇叭
    \item \textbf{音源设备}：专业麦克风、广播控制台、数字音频播放器
    \item \textbf{控制设备}：分区器、矩阵切换器、紧急呼叫器
\end{itemize}

\section{有线广播设计与安装}
\label{sec:wired_broadcast_design}

设计要点：

\begin{itemize}
    \item \textbf{功率计算}：根据扬声器数量和功率计算功放容量
    \item \textbf{线路设计}：选择合适的线缆规格和传输方式
    \item \textbf{分区规划}：合理划分广播区域
    \item \textbf{声学设计}：考虑声场均匀度和音质效果
\end{itemize}

安装注意事项：扬声器布局、线缆敷设、接地处理、防雷保护。
