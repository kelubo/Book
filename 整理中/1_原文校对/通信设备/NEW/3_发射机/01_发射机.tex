\chapter{发射机}
\label{chap:transmitter}

\section{发射机概述}
\label{sec:transmitter_intro}

发射机是一种将低频信号（如音频、视频）调制到高频载波上，并通过天线辐射到空间的电子设备。发射机是无线电通信系统的核心组成部分。

发射机的主要分类：
\begin{itemize}
    \item \textbf{按功率等级}：小功率发射机、中功率发射机、大功率发射机
    \item \textbf{按工作频段}：长波、中波、短波、超短波、微波发射机
    \item \textbf{按调制方式}：AM发射机、FM发射机、SSB发射机、数字调制发射机
\end{itemize}

\section{发射机基本原理}
\label{sec:transmitter_principle}

发射机的基本工作流程：

1. \textbf{信号源}：产生原始低频信号（音频、视频或数据）
2. \textbf{调制器}：将低频信号调制到高频载波上
3. \textbf{高频放大器}：放大调制后的高频信号
4. \textbf{天线}：将高频信号辐射到空间

发射机的核心指标包括输出功率、效率、频率稳定度、调制精度等。

\section{高频功率放大器}
\label{sec:rf_power_amplifier}

高频功率放大器是发射机的关键部件，用于将调制后的信号放大到足够的功率电平。

功率放大器的工作类别：
\begin{itemize}
    \item \textbf{A类放大器}：线性放大，效率低（约25-50\%）
    \item \textbf{B类放大器}：半波导通，效率较高（约78.5\%）
    \item \textbf{C类放大器}：小于半波导通，效率高（约80-90\%），但非线性
    \item \textbf{D类放大器}：开关工作，效率极高（>90\%）
\end{itemize}

功率放大器的主要技术指标：输出功率、效率、增益、线性度、谐波失真。

\section{调制器与解调器}
\label{sec:modulator_demodulator}

调制器用于将低频信号加载到高频载波上，常见调制方式：

\begin{itemize}
    \item \textbf{调幅（AM）}：载波振幅随调制信号变化
    \item \textbf{调频（FM）}：载波频率随调制信号变化
    \item \textbf{调相（PM）}：载波相位随调制信号变化
    \item \textbf{数字调制}：如ASK、FSK、PSK、QAM等
\end{itemize}

调制器的设计需考虑调制指数、带宽、线性度等因素。

\section{发射机设计与调试}
\label{sec:transmitter_design}

发射机设计要点：

\begin{itemize}
    \item \textbf{频率合成}：稳定的载波频率源
    \item \textbf{功率放大}：高效率、高线性度功率放大器
    \item \textbf{调制电路}：高质量的调制器
    \item \textbf{滤波网络}：减少谐波辐射
    \item \textbf{电源供应}：稳定、高效的电源
\end{itemize}

调试项目包括：功率测量、频率校准、调制深度调整、谐波抑制测试等。
