\chapter{发射机}
\label{chap:transmitter}

\section{发射机概述}
\label{sec:transmitter_intro}

发射机是一种将低频信号（如音频、视频）调制到高频载波上，并通过天线辐射到空间的电子设备。发射机是无线电通信系统的核心组成部分。

发射机的主要分类：
\begin{itemize}
    \item \textbf{按功率等级}：小功率发射机、中功率发射机、大功率发射机
    \item \textbf{按工作频段}：长波、中波、短波、超短波、微波发射机
    \item \textbf{按调制方式}：AM发射机、FM发射机、SSB发射机、数字调制发射机
\end{itemize}

\section{发射机基本原理}
\label{sec:transmitter_principle}

发射机的基本工作流程：

1. \textbf{信号源}：产生原始低频信号（音频、视频或数据）
2. \textbf{调制器}：将低频信号调制到高频载波上
3. \textbf{高频放大器}：放大调制后的高频信号
4. \textbf{天线}：将高频信号辐射到空间

发射机的核心指标包括输出功率、效率、频率稳定度、调制精度等。

\section{高频功率放大器}
\label{sec:rf_power_amplifier}

高频功率放大器是发射机的关键部件，用于将调制后的信号放大到足够的功率电平。

功率放大器的工作类别：
\begin{itemize}
    \item \textbf{A类放大器}：线性放大，效率低（约25-50\%）
    \item \textbf{B类放大器}：半波导通，效率较高（约78.5\%）
    \item \textbf{C类放大器}：小于半波导通，效率高（约80-90\%），但非线性
    \item \textbf{D类放大器}：开关工作，效率极高（>90\%）
\end{itemize}

功率放大器的主要技术指标：输出功率、效率、增益、线性度、谐波失真。

\section{调制器与解调器}
\label{sec:modulator_demodulator}

调制器用于将低频信号加载到高频载波上，常见调制方式：

\begin{itemize}
    \item \textbf{调幅（AM）}：载波振幅随调制信号变化
    \item \textbf{调频（FM）}：载波频率随调制信号变化
    \item \textbf{调相（PM）}：载波相位随调制信号变化
    \item \textbf{数字调制}：如ASK、FSK、PSK、QAM等
\end{itemize}

调制器的设计需考虑调制指数、带宽、线性度等因素。

\section{发射机设计与调试}
\label{sec:transmitter_design}

\subsection{发射机设计要点}

\begin{itemize}
    \item \textbf{频率合成}：稳定的载波频率源，可采用晶体振荡器、锁相环（PLL）或直接数字频率合成器（DDS）
    \item \textbf{功率放大}：高效率、高线性度功率放大器，需考虑散热设计
    \item \textbf{调制电路}：高质量的调制器，确保调制精度和线性度
    \item \textbf{滤波网络}：减少谐波辐射，满足电磁兼容（EMC）要求
    \item \textbf{电源供应}：稳定、高效的电源，考虑纹波和噪声抑制
    \item \textbf{保护电路}：过压、过流、过热保护
\end{itemize}

\subsection{频率合成技术}

频率合成器用于产生稳定、准确的载波频率。

\begin{itemize}
    \item \textbf{晶体振荡器}：高稳定性，固定频率输出
    \item \textbf{锁相环（PLL）}：通过反馈环路锁定频率，可实现频率捷变
    \item \textbf{直接数字频率合成器（DDS）}：数字方式生成频率，分辨率高、切换速度快
\end{itemize}

PLL的基本组成：参考振荡器、鉴相器（PD）、环路滤波器、压控振荡器（VCO）

\subsection{发射机指标测试}

调试项目包括：

\begin{itemize}
    \item \textbf{功率测量}：输出功率、效率
    \item \textbf{频率校准}：载波频率准确度和稳定度
    \item \textbf{调制参数}：调制深度、调制失真
    \item \textbf{谐波抑制}：二次谐波、三次谐波抑制比
    \item \textbf{杂散辐射}：非谐波杂散分量
    \item \textbf{邻道功率}：对相邻频道的干扰
\end{itemize}

\subsection{发射机效率计算}

发射机总效率：$\eta_{total} = \frac{P_{out}}{P_{dc}}$

其中，$P_{out}$为射频输出功率，$P_{dc}$为直流输入功率。

功率放大器效率：$\eta_{PA} = \frac{P_{RF}}{P_{dc-PA}}$

整体效率取决于功率放大器效率、调制器效率和电源效率等因素。
