\chapter{前言}

本书旨在介绍日本AV女优的行业生态、工作实况以及知名女优的信息。AV女优是一个常被社会误解的职业，实际上她们是具有专业素养的职场女性。

\mainmatter

\chapter{AV女优行业概述}

\section{行业规模与数据}

日本每年的A片产量约有1万部，目前约有2000名女优在这个产业中活动。这个数字虽然看起来不少，但其中大约有八成会在一年之内因为各种原因退出业界。

经纪公司通常分为大、中、小三种规模：
\begin{itemize}
\item 大型公司：女优人数超过100人
\item 中型公司：女优人数约50人
\item 小型公司：女优人数不到10人
\end{itemize}

全东京都内约有100间相关经纪公司，这些公司负责发掘、培训和推荐女优给各片商。

\section{女优的分类}

AV女优主要分为以下三种类型：

\subsection{单体女优}

单体女优是会被片商提拔成主角的顶级女优。她们的照片会出现在商品包装上，能够形成固定的粉丝团。由于这类女优在业界被视为高等级族群，因此大多会与特定片商签下专属契约。

\subsection{企划单体女优}

企划单体女优有着单体女优的超高人气，但没有和片商签下专属契约。由于她们不受契约约束，可以自由接演各片商的A片，因此作品数量往往能够迅速飙升。

\subsection{企划女优}

企划女优与单体女优相比更难自成气候，通常都是接演多位女优同时登场的作品，如《女子性爱温泉旅行大会》等，有时她们的艺名甚至不会印在商品包装上。在2000名现役AV女优中，大约有九成都是企划女优。

\chapter{AV女优的工作实况}

\section{拍摄日程}

A片的拍摄日程通常非常紧凑。以溜池五郎导演的工作现场为例：
\begin{itemize}
\item 早上：全体人员集合
\item 下午1点前：拍摄封面照等相关摄影
\item 下午1点后：正式开拍
\item 晚餐时间：工作人员轮流用餐
\item 必须在9小时内完成所有场景
\end{itemize}

\section{工作强度}

AV女优的拍摄工作堪称重体力劳动。她们需要在数小时内频繁演出性爱场面，而且每次性爱场面都得持续一小时以上。

在低预算的拍片现场，工作强度相对较低，一部片的预算通常不超过50万日圆。而在高预算的制作中，女优一整天的片酬可能高达20万至30万日圆。

\section{片酬体系}

AV女优的片酬因级别和类型而异：
\begin{itemize}
\item 知名单体女优：单日片酬可达20-30万日圆甚至更高
\item 企划女优：片酬相对较低
\item 刚出道的新人：片酬通常在较低水平
\end{itemize}

需要注意的是，女优的片酬通常由片商支付给经纪公司，再由经纪公司转发给女优。

\chapter{知名女优介绍}

\section{川奈真理子}

川奈真理子是溜池五郎导演的妻子，也是业界知名的熟女系女优。

\subsection{基本信息}
\begin{itemize}
\item 年龄：1971年出生（拍摄《义母，真理子34岁》时31岁）
\item 出道时间：1999年
\item 退役时间：2004年
\item 代表作品：《义母，真理子34岁》
\end{itemize}

\subsection{从业经历}

川奈真理子在1999年以熟女身份出道，凭借《义母，真理子34岁》一炮而红。这部作品不仅确立了溜池五郎的导演风格，也开创了日本AV界的"美熟女"时代。

从业5年间，她共演出约400部作品。2004年与溜池五郎结婚并生下一子后选择退役。

\subsection{退役后发展}

退役后，川奈真理子转型为作家，于2011年出版长篇小说《继母的香艳》，成功从AV女优转型为情色小说家。后来又发表《禁忌——继母与婶婶与我》等多部作品，在限制级文学领域占有一席之地。

\section{米仓真耶}

米仓真耶是溜池五郎个人专属品牌的第二位女优，以肉食系熟女的形象著称。

\subsection{特征}
\begin{itemize}
\item 外表：看起来像有黛比夫人威严和妖艳气息的20多岁女性
\item 身材：略显丰腴
\item 性格：爽朗快乐，面试时经常"啊哈哈哈哈哈"地咧嘴大笑
\end{itemize}

\subsection{业界地位}

米仓的性爱技巧据称凌驾于所有女优之上，被誉为"肉食系熟女天后"。她是"熟女+肉食系"这一全新女优属性的代表人物。

\section{木下由香里（化名）}

木下由香里是2001年《人妻羞耻部屋》系列作品的主演，当时31岁，是一位育有孩子的母亲。

\subsection{从业原因}

自从当母亲以来，无论何时何地都被称为"某某太太"和"某某妈妈"，这对女性来说等于是否定了她作为女性的身份。妊娠纹、胸部下垂、乳头变黑等产后变化让她深感困扰。

在溜池五郎的鼓励下，她意识到这些身体变化正是成熟女性的魅力所在，因此决定出演这部作品。

\section{石本佐江（化名）}

石本佐江于2008年由片商负责人带来面试，当时46岁。虽然还未撤掉丈夫的户籍，但双方早已长年分居。

她的从业动机与经济压力有关。44岁时与儿子的18岁同学发生婚外情，这段经历让她重新审视自己的人生。

\section{真田有美（化名）}

真田有美当时35岁，有着鸭子嘴和纤瘦身材，指甲上华丽的彩绘让人看不出是家庭主妇。

她从业的动机是报复丈夫的外遇。在性方面与丈夫产生障碍多年后，她选择用这种方式来宣泄愤怒。

\chapter{美熟女文化}

\section{熟女的定义}

在AV业界，通常把27岁以上的女优称为"熟女"。这个定义在2000年前后开始流行，溜池五郎导演是这一类型作品的先驱。

\section{美熟女的兴起}

1999年，溜池五郎导演的《义母，真理子34岁》发售当时造成热卖，这部作品确立了"美熟女"这个新属性。

在此之前，27岁以上的女优在业界被认为没有任何商品价值，溜池五郎的选角方式在当时被认为是违背常理的异例，甚至有业界人士嘲讽他是"脑子不正常了"。

\section{熟女与社会经济}

熟女作品的兴起与日本"失落的10年"（1991-2002）经济低迷期有着密切关系。

在经济不景气的环境下，女性不得不更加努力地满足男性需求。传统的"男人送钱、送车、送化妆品"的模式转变为"女性若不在性爱上满足男性，就无法获得想要的东西"的时代。

\chapter{女优的生活与心理}

\section{职业认同}

随着行业的发展，AV女优越来越将这份工作视为普通的职业。她们会敬业地完成工作，就像任何其他职业一样。

溜池五郎有拍完片送花给女优致谢的习惯。早期女优们常感动地流泪，后来她们越来越习惯带着微笑接过花束——这表明她们已经认真地将拍A片当成工作看待。

\section{隐私保护}

AV业界对女优的个人信息保护非常重视。"父母抓包"和"男友抓包"是所有女优最担心的事情。

业界会通过各种方式保护女优的身份，包括使用艺名、模糊化个人信息等。这也是为什么每年能有将近2000位女优愿意出道的原因之一。

\section{退役转型}

AV女优退役后的出路多样：
\begin{itemize}
\item 结婚生子，过普通生活
\item 转型为作家（如川奈真理子）
\item 继续在演艺圈发展
\item 转行做其他工作
\end{itemize}

\chapter{业界运作机制}

\section{销售模式演变}

AV作品的销售方式经历了多次变革：
\begin{itemize}
\item 早期：录像带出租
\item 后来：DVD直营店销售
\item 现在：网络下载和流媒体
\end{itemize}

\section{制作流程}

一部A片从企划到上架需要经过以下环节：
\begin{enumerate}
\item 企划阶段：构思新题材和卖点
\item 选角：挑选合适的女优
\item 拍摄：通常需要一天完成所有场景
\item 后期制作：剪辑、配音、特效处理
\item 包装设计和发行销售
\end{enumerate}

\section{经纪公司}

经纪公司是连接女优和片商的重要桥梁。大型经纪公司如SOD Create拥有数百位女优资源，能够为片商提供丰富的选择。

\chapter{结语}

AV女优是一个常被社会误解的职业。她们中的大多数人都经济独立，对工作敬业的态度堪称职业级水准。

当AV女优离开萤幕焦点后，也一样会有家人、情人、丈夫以及自己的孩子相伴，其中更不乏是拥有旺盛生命力的坚强女性。

本书希望通过介绍AV女优的真实工作和生活，让社会对这个行业有更多的理解和尊重。
